## macOS hexo 使用

node 环境安装

```shell
brew install nodejs
npm -g install n
sudo n latest
sudo npm -g i npm
npm -g i hexo
```

ftp 安装

```shell
# 这里面包括了一系列的网络工具包, 有 dnsdomainname、ftp、telnet 等
brew install inetutils
```
